# AI Development Partner Guides

**Purpose:** This folder contains guidance documents designed for AI development tools (Cursor AI, Claude Projects, and similar) to reference when generating, reviewing, or validating Refuge House documentation.

**Audience:** AI systems and developers working with AI-assisted document generation

## Usage Notes

- These guides synthesize requirements from multiple regulatory sources into AI-consumable formats
- All T3C Blueprint references are to the **October 2025** edition (515 pages)
- Guides include specific page references for traceability
- Evidence standards and integrity rules prevent AI fabrication or assumption
